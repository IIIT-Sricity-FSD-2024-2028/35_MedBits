# 35_MedBits
## Domain: HealthCare Administration (Non-Clinical)
## Problem Statement: Patient Appointment and Registration Management System
The system aims to streamline and digitalize the process of patient registration, appointment scheduling, and consultation management within a healthcare facility.  
This system provides a platform where:  
- Patients can register and book appointments efficiently.  
- Front desk staff can manage walk-ins, scheduling and billing accurately.  
- Doctors can access patient records, document consultations and manage follow-ups seamlessly.
